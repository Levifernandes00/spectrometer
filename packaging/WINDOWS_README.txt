SPECTROMETER WINDOWS QUICK START
================================

1) Unzip this package into a permanent folder, for example:
   C:\Spectrometer
   or
   %LOCALAPPDATA%\Spectrometer

2) Edit config.example.json with your values, then save it as:
   config.json
   in the SAME folder as spectrometer.exe.

3) Run spectrometer.exe once to validate your setup.

Start with Windows (Startup folder)
-----------------------------------

1) Press Win + R and run:
   shell:startup

2) Create a shortcut to spectrometer.exe in that Startup folder.

3) Reboot or sign out/in to verify automatic startup.

Notes
-----

- The application expects config.json next to spectrometer.exe.
- If config.json is missing on first run, the app may create one from template.
